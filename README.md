# ShareIt

ShareIt — backend-сервис для аренды и обмена вещами между пользователями.  
Пользователи могут добавлять вещи, бронировать их на определённые даты, оставлять комментарии и управлять своими бронированиями.

Проект разработан в рамках обучения в Яндекс Практикуме.

---

## Функциональность

### Пользователи
- создание и редактирование пользователей;
- получение информации о пользователе.

### Вещи
- добавление вещей;
- обновление информации о вещах;
- поиск доступных вещей;
- просмотр списка вещей владельца.

### Бронирования
- создание бронирования;
- подтверждение или отклонение владельцем;
- просмотр текущих и прошлых бронирований;
- фильтрация бронирований по статусу.

### Комментарии
- возможность оставлять комментарии после завершённой аренды.

## Запуск проекта

### Требования

- Java 17+
- Maven
- PostgreSQL
- Docker (опционально)

### Сборка проекта

```bash
mvn clean install
```

### Запуск приложения

```bash
mvn spring-boot:run
```

## API

Основные endpoints:

### Users
- `POST /users`
- `GET /users/{id}`
- `PATCH /users/{id}`

### Items
- `POST /items`
- `GET /items/{id}`
- `GET /items`
- `GET /items/search`

### Bookings
- `POST /bookings`
- `PATCH /bookings/{id}`
- `GET /bookings`
- `GET /bookings/{id}`
